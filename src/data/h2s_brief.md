## 11. Report writing rules

1. **Voice:** Direct, agency-internal. Plain English, no jargon. Never say "stakeholders" — say "the team".
2. **WoW only** in Phase 1. Don't mention MoM or daily comparisons unless asked.
3. **Explain WHY, not just WHAT.** A 40% CPM jump needs a hypothesis (competitor auction pressure? creative fatigue? holiday?). State it as a hypothesis if you can't confirm it.
4. **Lead with anomalies.** If something moved more than 2σ from the trailing 4-week mean, that's the headline.
5. **End with concrete actions.** "Recommended actions next week" must be 3–5 numbered items, each tied to a specific data point above.
6. **Don't hedge artificially.** If the data says Meta drove 70% of revenue this week, say so. Don't add empty caveats.
7. **No emojis. No headers like 🎯 or 🚀.** Plain markdown only.
8. **Currency:** All amounts in INR (₹). Format ≥1 lakh as "₹1.2L", ≥1 crore as "₹1.2Cr".
9. **Audience: internal only Phase 1** — Digitally Next team + H2S COO. You can be blunt. You can name underperforming creatives. Don't worry about client diplomacy yet.

## 12. The "Power of Five" report structure

Every weekly report has **9 fixed sections**, in this order, no skipping:

1. **Headline KPIs** — Revenue, orders, blended CAC, blended ROAS, prepaid %, RTO % (each with WoW delta).
2. **By geography** — Top 6 primary states (TN/GJ/AP/MH/TG/PY) with ROAS + CAC + order volume. Flag any state that dropped >20% WoW.
3. **By channel** — Meta / Google / Amazon / Flipkart / Direct. Each with ROAS + revenue share + WoW delta.
4. **Meta by quadrant** — All 4 quadrants with spend share, ROAS, and best/worst creative this week.
5. **LTV cohorts** — All 4 cohorts: size, replacement rate (replace_9_12 cohort), WoW change.
6. **Funnel health** — GA4 sessions → add-to-cart → checkout → purchase rates. Clarity rage/dead clicks. WhatsApp inquiry conversion if data is available.
7. **Search intent (GSC)** — Top 10 converting queries this week. Then top 10 high-impression queries with low CTR or no conversions (these are content gaps).
8. **Anomalies** — Bulleted list. For each: metric, magnitude, plain-English hypothesis, suggested investigation.
9. **Recommended actions next week** — 3–5 numbered items, each citing the section above that motivates it.

Never invent data. If a section has no data (e.g. GSC integration is still being set up), write "*No GSC data this week — integration in progress.*" and skip to the next section.
